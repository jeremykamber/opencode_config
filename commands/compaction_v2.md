## The AI Agent Prompt

**Role:** You are an expert Senior Frontend Developer and UI/UX Designer.

**Objective:** Build a modern, high-converting, and visually stunning landing page for [INSERT PRODUCT/SERVICE NAME] that includes a mailing list signup form.

**Technical Requirements:**
* **Framework:** React with Next.js (App Router).
* **Styling:** Tailwind CSS.
* **Icons:** Lucide-React.
* **Animations:** Framer Motion (for smooth fades and entrance movements).
* **Form Handling:** Create a controlled component for the email input with basic client-side validation.

**Design Aesthetics:**
* **Style:** [Choose one: Minimalist / Dark Mode Tech / Playful & Colorful / Professional Corporate].
* **Layout:** A "Bento Box" style or a clean centered hero section.
* **Visuals:** Use high-quality placeholders for images or decorative abstract SVG shapes. Ensure generous whitespace and modern typography (Sans-serif).

**Page Structure:**
1.  **Navigation:** Simple logo on the left, "Features" and "Pricing" links (anchor tags), and a "Join Now" CTA button on the right.
2.  **Hero Section:** * A catchy H1 headline (e.g., "The future of [X] is here").
    * A subheadline explaining the value proposition.
    * An **Email Signup Form** (Input field + "Subscribe" button) prominently displayed.
3.  **Social Proof:** A small "Trusted by 1000+ users" badge or a row of grayscale logos.
4.  **Features Grid:** A 3-column grid showing three key benefits with icons.
5.  **Footer:** Copyright, social links, and a secondary signup prompt.

**Specific Instructions:**
* Ensure the design is **fully responsive** (mobile-friendly).
* Add a "Success" state for the email form (e.g., replace the form with a "Thanks for joining!" message upon submission).
* Use a sophisticated color palette: Primary color [e.g., Indigo #6366f1], Background [e.g., Slate #0f172a].
